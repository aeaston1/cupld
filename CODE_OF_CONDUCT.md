# Contributor Code of Conduct

## Our Standards

We want `cupld` to be a respectful, welcoming, and harassment-free place.

Good conduct includes:

- Using welcoming and inclusive language.
- Respecting different perspectives and backgrounds.
- Refraining from harassment, insults, or personal attacks.
- Keeping discussion focused on technical merit and project quality.

## Unacceptable Behavior

- Harassment, discriminatory remarks, or derogatory language.
- Public shaming, threats, or targeted harassment.
- Sharing private information without consent.
- Publishing release or security information with intent to harm users.

## Reporting

## Scope

This code applies to project spaces, including repository discussions, issue threads,
pull requests, release channels, and project-linked communication channels.
